Downloading the LDM-McIDAS Package

LDM-McIDAS package is freely available as program source and as binary distributions for a wide range of platforms supported by the UPC.

Please read the following notes before downloading the package.

  • If you want to build the LDM-McIDAS decoders from source, you will need to be a registered McIDAS MUG Member site since several of the decoders are wrappers around copyrighted McIDAS-X code.
  • Unidata McIDAS-X is available to universities worldwide. For more information, contact <support-ldm-mcidas AT unidata.ucar.edu>.
  • We suggest building/installing LDM-McIDAS under the ldm account.

Source & Binary Distributions

The most current current source and binary distributions of LDM-McIDAS is available in the compressed tar file from our downloads site.

Information on building ldm-mcidas from source is found in the next section, Building and Installing.

Download Procedure

Since the LDM-McIDAS decoders are used almost exclusively with the LDM, we strongly suggest installing them in the ldm account. Login as the user ldm user, ands type the following:

      cd ~ldm
      mkdir ldm-mcidas
      cd ldm-mcidas

Then download the LDM-McIDAS source or binary distribution compressed tar file to this directory>
