LDM-McIDAS Releases

LDM-McIDAS decoders are updated infrequently. Their release typically corresponds to support of new computing platforms by the UPC, significant updates in decoder functionality (bugfixes or new decoders), when programmatic interfaces for packages that the decoders depend on change, or when imagery in the Unidata-Wisconsin datastream (IDD stream UNIWISC which is equivalently known as MCIDAS).

The version numbering used for LDM-McIDAS is intended to reflect the release of Unidata McIDAS-X against which it was built. For instance, LDM-McIDAS Version 2008 is compatible with Unidata McIDAS-X Version 2008; etc.
